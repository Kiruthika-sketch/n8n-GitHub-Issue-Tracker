# 🐙 GitHub Issue Tracker Automation (n8n)

## 📌 Overview

This project automates GitHub issue management using n8n. It creates and manages GitHub issues through workflow automation, reducing manual effort and improving project tracking.

## 🚀 Features

- Creates GitHub issues automatically
- Connects with GitHub API
- Supports custom issue title and description
- Easy to extend for project management

## 🛠️ Tech Stack

- n8n
- GitHub API
- HTTP Request Node
- Trigger Node
